CI / CD / DevOps Pipeline Tools 大亂鬥

華山論劍，經驗交流

DevOps Taiwan 社群目前平均每個月會舉辦一次社群活動，邀請社群講者分享各種與 DevOps 相關的講題，其中自 2017 年開始每年皆會舉辦一場名為「大亂鬥」的中型活動。2017 的大亂鬥主題為「Configuration Management Tools 大亂鬥」、2018 主題為「Monitoring Tools 大亂鬥」，而今年的主題則鎖定為「CI / CD / DevOps Pipeline Tools 大亂鬥」。

「大亂鬥」活動設定的目標是提供如同「華山論劍」一般的空間，讓與會者可以在一場活動中迅速地認識各家工具及解決方案各自的優異之處，為企業、組織或團隊的工具決策提供更多參考資訊， 同時藉由一次匯聚來自不同供應商、社群、企業或組織的講者及與會者，各自可以根據自身所運用這些工具的實戰經驗彼此進行深度的經驗交流。

活動詳情

活動地點、日期、議程

地點

達文西會議空間 - 羅馬廳（台北市南京東路二段6號6樓）

日期與時間

2019 年 4 月 20 日，13:30 ~ 18:00（13:30 ~ 14:00 報到）

議程

上半場：解決方案及工具之火力展示（上台順序依工具之名稱排序）

每位講師擁有 20 分鐘，可以展示一條完整的 CI / CD Pipeline，並在時間內盡力展現該解決方案的優點。

  • AWS DevOps Tools (CodePipeline...及其他工具)
  • Azure DevOps
  • Circle CI
  • Drone CI
  • GCP DevOps Tools (Cloud Build...及其他工具)
  • GitLab CI
  • Jenkins
  • JetBrains TeamCity

下半場：CI / CD / DevOps Pipeline 之綜合座談暨實戰經驗交流

由所有的講師、受邀之資深從業者及現場參加者，共同進行綜合座談，分享各自在 CI / CD 的實戰經驗，以及回答現場參與者的各項提問。
